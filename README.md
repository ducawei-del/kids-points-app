# 好習慣存摺挑戰 App

這是可直接部署到 Vercel 的版本。

## 本機啟動

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## 上線到 Vercel

1. 到 Vercel 註冊並登入
2. 點 `Add New Project`
3. 選擇 `Import` 或直接上傳這個專案
4. Framework 選 `Vite`（通常會自動辨識）
5. 點 `Deploy`

## 功能說明

- Yoi / Sunny 點數管理
- 加點 / 扣點 / 兌換獎勵
- 每週小目標顯示
- 可上傳本機大頭照
- 本機 localStorage 保存資料
- 可產生分享連結（不包含上傳的大頭照）
